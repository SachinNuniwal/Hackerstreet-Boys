# React Hackathon – Mini Event Manager

## 🎯 Objective
Build a Mini Event Manager using React.

## ⏱ Duration
3 Hours

## ✅ Mandatory Features
- Login Page (dummy authentication allowed)
- Dashboard
- Add Event (Title, Date, Description)
- Display Event List
- Edit Event
- Delete Event
- Store data using localStorage
- Responsive UI
- Clean folder structure

## ⭐ Bonus
- Search functionality
- Sorting by date
- Dark mode
- Form validation

## 🚫 Rules
- No backend
- No database
- Only React
- No copying from internet

## 📦 Submission
- Push code regularly
- Final commit before deadline
- Optional: Deploy on Vercel
